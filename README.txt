ibc-disease-progression
=======================

Tools and analyses related to using a time series of relevant lab values for a cohort of interest to predict clinical outcomes

ckd.py: Runs the entire pipeline for predicting dialysis or kidney transplant in a cohort of patients with Stage IV chronic kidney disease

patient_stats.py: A utility to calculate statistics on groups of patients

build_training_data.py: Build training examples from lab data patient stats

Tests
=====

To run tests, type in the command line:

nosetests
